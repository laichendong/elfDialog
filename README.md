elfDialog 一个简易对话框实现
==============================

#选项

##type
**dataType:**`String` 可选值有:`"dialog"	"alert"	"confirm"`	**default:**`"dialog"`

类型。不同的类型对话框会有不同的按钮和默认的行为

----

##width
**dataType:**`int`	**default:**`0`

宽度。

----

##height
**dataType:**`int`	**default:**`0`

高度。

----

##title
**dataType:**`String`	**default:**`""`

标题。
